Преимущества участия в студенческих проектах

Обучение

Работа в проекте предваряется вводным курсом лекций, посвящённых специфике выбранного проекта. По ходу проекта проводятся семинары и презентации.

Совмещение с учёбой

Набор в студенческие проекты возобновляется с началом каждого учебного семестра. В каждом проекте участвуют около десяти студентов. Проекты длятся от одного до двух семестров.

Результат

Участвуя в студенческих проектах, вы получаете:
опыт практической работы в IT-компании
дополнительные теоретические знания
возможность получить приглашение на работу.
